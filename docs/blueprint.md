# **App Name**: Artisan Showcase

## Core Features:

- Product Display: Showcase candles and coasters in an appealing card format, including image, name, description, and price.
- Admin Panel: A password protected admin panel with username 'admin' and password '0123456789' for managing products.
- Product Upload: Form inputs in the admin panel for uploading product details including name, description, price, and CDN image link.
- Product Management: Ability to add, edit, and delete products via the admin panel.

## Style Guidelines:

- Primary color: Soft terracotta (#E07A5F), evokes handmade artisanry.
- Background color: Light beige (#F8F0E3), to showcase products.
- Accent color: Warm gold (#D4A373), draws eye to important UI elements.
- Body and headline font: 'Alegreya', serif, elegant with intellectual contemporary feel.
- Minimal and modern line icons for navigation and calls to action.
- Clean and minimalist layout with generous spacing to highlight the products.
- Subtle fade-in animations on product cards as the user scrolls.
